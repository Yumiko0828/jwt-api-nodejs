# Jwt API Nodejs

## Requirements:

- MongoDB

## Installation:

clone this repository

```
git clone https://github.com/Yumiko0828/jwt-api-nodejs.git
```

### Install dependencies

yarn:

```
yarn install
```

npm:

```
npm install
```

## Build:

yarn:

```
yarn run build
```

npm:

```
npm run build
```
